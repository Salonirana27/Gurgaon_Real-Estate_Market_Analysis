# Gurgaon_Real-Estate_Market_Analysis# 🏠 Gurgaon Real Estate Market Analysis

A Python-based Data Analytics project that explores the Gurgaon (Gurugram) real estate market using Exploratory Data Analysis (EDA) and data visualization. The project analyzes property prices, locality trends, builder performance, and other important real estate factors to answer real-world business questions.

---

## 📌 Project Overview

This project aims to analyze Gurgaon real estate data and generate meaningful insights that can help buyers, investors, and real estate professionals make informed decisions.

## 📊 Business Questions Answered

This project answers the following business questions:

1. Which is the costliest flat in the dataset?
2. Which locality has the highest average price?
3. Which locality has the highest rate per square foot?
4. Do ready-to-move properties cost more than under-construction properties?
5. Do RERA-approved properties command a price premium?
6. How does area (sqft) impact property price?
7. Which BHK configuration is the most expensive on average?
8. Which property type (Apartment, Floor, Plot) is the costliest?
9. Do certain builders or companies consistently price higher?
10. Are larger homes always more expensive per square foot?
